# sql-hospital-management-system
A simple PostgreSQL-based Hospital Management System to manage patients, doctors, appointments, and billing using SQL.

A PostgreSQL-based Hospital Management System to manage:

- Doctors
- Patients
- Appointments
- Billing

## Features
- Foreign key relationships
- JOIN queries
- Revenue calculation
- Real-world database structure

## Tech Used
- PostgreSQL
- SQL (DDL, DML, JOIN, GROUP BY)
